# 研发费用加计扣除风险管理体系 — Phase 2（M7–M12）实施计划

> **For 项目实施团队**：本计划是 Phase 1 收官后的下一阶段，6 个月内完成从"体系跑通"到"经得起年度汇算清缴"。任务以 checkbox (`- [ ]`) 跟踪。Phase 3（M13–M24）将在 Phase 2 收官后另立 plan。

**Goal:** 在 6 个月内（M7–M12）完成委外 / 合作研发治理、"其他相关费用"5% 限额管控、与高新认定口径并行比对、年度专项自查、首次年度汇算清缴申报，体系经得起税务稽查 + 年报审计 + IPO 披露三重检验。

**Architecture:** 在 Phase 1 已稳定运行的"治理 + 判定 + 证据 + 内控"四层基础上，叠加 Phase 2 三个关键模块：① 委外 / 合作合同治理 ② "其他相关费用"管控 ③ 年度自查与汇算清缴。

**Tech Stack:** 飞书多维表格（既有）+ ERP 财务系统（既有）+ 外部税务师事务所辅导（新增）。
