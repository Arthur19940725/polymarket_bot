# 10 类档案清单与索引

| 项 | 内容 |
|---|---|
| 文档版本 | v1.0 |
| 形式 | 飞书多维表格「档案索引」+ Excel 副本 |
| 生效日期 | 2027-04-15 |
| 上级文件 | 《留存备查档案管理办法》|
| 维护方 | 加计扣除办公室 |

> **使用说明**：本文件是 10 类档案的**完整清单**，包含每类档案的：① 文件命名规则 ② 来源部门 ③ 飞书云路径 ④ 纸质归档位置 ⑤ 责任人 ⑥ 收集时点。索引表本身也作为档案的一部分，每年随档案包一起归档。

## 第一章 索引表结构（飞书多维表格）

### 字段定义

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 索引号 | 自动编号 | ✅ | 格式 `INDEX-YYYY-NNNN` |
| 档案类别 | 单选 | ✅ | 1-10 类 |
| 子类别 | 文本 | ✅ | 如"立项申请书"、"科技局登记证明" |
| 关联项目编码 | 文本 | | 如 RD-2026-007 |
| 文件名 | 文本 | ✅ | 飞书云规范命名 |
| 飞书云路径 | 链接 | ✅ | 完整路径 |
| 纸质柜号 | 文本 | | 如 A-3-2-5（柜-排-层-卷）|
| 纸质份数 | 数字 | ✅ | 一般 2 份 |
| 收集时点 | 日期 | ✅ | 入档日期 |
| 来源部门 | 单选 | ✅ | 研发 / HR / 财务 / 法务 / 加计办 / 内审 |
| 责任人 | 文本 | ✅ | 工号 + 姓名 |
| 密级 | 单选 | ✅ | 公开 / 内部 / 机密 |
| 保管期截止 | 日期 | ✅ | 入档日 + 10 年 |
| 调阅次数 | 数字 | ✅ | 累计 |
| 状态 | 单选 | ✅ | 在档 / 已销毁 / 已迁出 |
| 备注 | 长文本 | | |

### 第 1 类：研发项目立项书

**政策依据**：40 号公告（一）
**主管部门**：加计扣除办公室
**收集时点**：每个项目立项（G1）通过后 7 日内

| 子类别 | 文件 | 命名规则 | 数量 / 年 |
|--------|------|----------|-----------|
| 立项申请书 | PDF | `RD-YYYY-NNN-立项申请书-v1.0.pdf` | 每个 A/B 类项目 1 份 |
| 技术不确定性说明 | PDF | `RD-YYYY-NNN-技术不确定性说明.pdf` | 含在立项书内 |

## 第三章 索引表使用

### 第 2 条 索引表查询

#### 方式 1：按项目编码

在飞书表「档案索引」筛选"关联项目编码 = RD-YYYY-NNN"，可看到该项目全部档案的位置。

#### 方式 2：按档案类别

按"档案类别"分组，看每类的完整清单。

#### 方式 3：按时间范围

按"收集时点"区间筛选，看某月 / 某年的档案概况。

### 第 3 条 索引表的更新

- **每月归集后**：加计办主任更新本月新增档案条目
- **年度档案打包时**：完整核对索引表与实际档案的一致性
- **每次调阅时**：自动记录调阅次数

### 第 4 条 索引表的导出

每年汇算清缴申报后 30 天内：
- 导出当年完整索引表为 Excel
- 与档案包一起入库
- 纸质打印 2 份（加计办留 1 份，纸质档案库留 1 份作为内置目录）
