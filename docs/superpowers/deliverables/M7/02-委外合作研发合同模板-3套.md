# 委外 / 合作研发合同模板（3 套）

| 项 | 内容 |
|---|---|
| 文档版本 | v1.0 |
| 适用日期 | 2027-01-15 起 |
| 维护方 | 法务部 + 加计扣除办公室 |
| 使用规则 | **三套模板二选一执行，禁用旧版本** |
| 上级文件 | 《委外 / 合作研发合同审核 SOP》《知识产权归属说明书》 |

> **使用说明**：本文件包含三套模板：
> - **模板 A**：委托境内单位 / 个人研发（最常用）
> - **模板 B**：委托境外单位 / 个人研发（受三分之二限制）
> - **模板 C**：合作研发协议（双方共担风险共享成果）
>
> 业务部门起草合同时，须使用三选一模板。**禁止自行起草不基于模板的合同**。

# 模板 A：境内委托研发技术合同（标准版）

```
                  境内委托研发技术合同

甲方（委托方）：__________
                法定代表人：__________
                地址：__________
                统一社会信用代码：__________

乙方（受托方）：__________
                法定代表人：__________
                地址：__________
                统一社会信用代码：__________

鉴于甲方需就（项目名称：__________）开展技术研究开发活动，但
受自身研发资源限制，需委托具有相应研发能力的乙方承担本项目
研发工作；乙方具备相应技术能力与研发资质，同意接受甲方委托。

经双方友好协商，依据《中华人民共和国民法典》《中华人民共和国
科学技术进步法》及国家相关税收政策，达成本合同如下：

═══════════════════════════════════════════════════════════════

第一条  研发内容

1.1 项目名称：__________
1.2 研发目标：__________
1.3 主要研发内容（≥200 字客观描述，明确技术不确定性所在）：
    __________
1.4 研发成果形式：☐软件代码 ☐技术方案 ☐专利 ☐论文 ☐其他__________

第二条  技术指标与验收标准

2.1 关键技术指标（须可量化）：
    指标 1：__________ 目标值 __________
    指标 2：__________ 目标值 __________
2.2 验收方式：
    ☐ 阶段性验收（按里程碑）
    ☐ 总体验收（项目结束后）
    ☐ 混合验收
2.3 验收依据：__________

第三条  研发周期与里程碑

3.1 起止时间：自 YYYY-MM-DD 至 YYYY-MM-DD，共 __ 个月
3.2 里程碑：
    M1（YYYY-MM-DD）：__________
    M2（YYYY-MM-DD）：__________
    M3（YYYY-MM-DD）：__________

第四条  交付物清单

| 交付物 | 形式 | 交付时点 |
|--------|------|----------|
| 技术方案文档 | PDF | M1 |
| 源代码 | Git 仓库交接 | M3 |
| 测试报告 | PDF | M3 |
| 用户手册 | PDF / 在线 | M3 |
| ... | | |

第五条  ⭐ 知识产权归属（不可修改）

5.1 本合同项下产生的全部研发成果，包括但不限于发明专利、
    实用新型、外观设计、软件著作权、集成电路布图设计、商业
    秘密、技术秘密、技术资料、研究报告、源代码、设计文档等
    （以下统称"研发成果"），其知识产权（含申请权与申请后取
    得的权利）归甲方所有。

5.2 乙方应在合同履行过程中和合同终止后协助甲方办理知识产权
    的申请、登记、变更、维护等手续，相关费用由甲方承担。

5.3 乙方不得就本合同项下研发成果以自己名义申请知识产权或
    向任何第三方转让、许可。

5.4 乙方在履行本合同过程中使用的乙方既有技术成果（"背景技
    术"），其知识产权仍归乙方所有；甲方在本合同项下及为使
    用研发成果，享有免费的、永久的、不可撤销的使用权。

第六条  ⭐ 研发活动性质声明（不可修改）

6.1 双方确认，本合同项下的工作属于《财政部、国家税务总局、
    科技部关于完善研究开发费用税前加计扣除政策的通知》（财
    税〔2015〕119 号）所定义的"研究开发活动"，具有以下特征：

    (1) 为获得科学与技术新知识、创造性运用科学技术新知识、
        或实质性改进技术、产品、服务、工艺；
    (2) 具有明确的技术目标与技术不确定性；
    (3) 不属于该文件所列负面清单。

6.2 甲方承诺：本项目已完成内部立项程序，立项申请书（含技术
    不确定性说明）已归档。

6.3 乙方了解上述事项，并配合甲方提供必要的研发活动证明材
    料，包括但不限于阶段报告、技术方案、测试数据、研发人员
    名单等。

6.4 双方同意：本合同将在签订后 30 日内由甲方报主管科技行政
    主管部门进行技术合同登记。乙方应予以配合。

第七条  合同金额与付款

7.1 合同总金额：人民币（大写）__________ 元（小写：__________）
    含 6% 增值税。

7.2 付款节点：
    ┌──────────────────┬────────┬────────┐
    │       节点        │ 比例    │ 金额   │
    ├──────────────────┼────────┼────────┤
    │ 合同签订后 7 日内 │ 30%    │        │
    │ M2 里程碑验收后    │ 30%    │        │
    │ 项目总体验收后    │ 30%    │        │
    │ 质保期结束后       │ 10%    │        │
    └──────────────────┴────────┴────────┘

7.3 付款条件：乙方提供增值税专用发票，发票内容须明确为
    "技术开发服务"或"技术开发费"，**不得开具为"技术服务费"
    或"咨询服务费"**。

7.4 甲方付款方式：银行转账。

第八条  研发失败的风险承担

8.1 双方确认，本合同项下研发活动存在技术不确定性，可能出现
    无法实现预期技术目标的情形。

8.2 若乙方按合同约定善意履行研发义务，但因客观技术原因未能
    达到第二条约定的技术指标，乙方可不承担违约责任，甲方仍
    应按本合同支付已发生的合理研发成本（具体金额由双方协商
    确定）。

8.3 若乙方未能勤勉尽责履行研发义务（如未投入约定的研发人
    员、未按计划开展研发工作等），属于乙方违约，按第十一条
    处理。

第九条  研发人员与资源投入

9.1 乙方承诺投入的核心研发人员名单（附件 1），不得擅自变更。
    主要人员变更须经甲方书面同意。

9.2 乙方承诺投入的研发资源（设备、场地、数据等），见附件 2。

9.3 乙方应建立研发工时台账，配合甲方进行必要的工作量核查。

第十条  保密

10.1 保密信息范围：双方在本合同履行过程中知悉的所有未公开
     信息（包括但不限于技术信息、商业信息、研发数据等）。

10.2 保密期限：自本合同签订之日起至研发成果公开发布之日止，
     且不少于 5 年。

10.3 违反保密义务的，违约方应赔偿守约方因此遭受的全部损失。

第十一条  违约责任

11.1 任何一方违反本合同约定，应承担相应违约责任：
     - 未按时交付：每延迟 1 日按合同总额 0.5%承担违约金，
       累计不超过合同总额 10%；
     - 重大违约（无法继续履行）：违约方退还已收款项 + 赔偿
       守约方实际损失。

11.2 不可抗力：因不可抗力导致合同无法履行的，双方互不承担
     违约责任，但应及时通知对方并协商后续处理。

第十二条  争议解决

12.1 因本合同发生争议的，双方应首先友好协商。

12.2 协商不成的，由甲方所在地有管辖权的人民法院诉讼解决。

12.3 本合同适用中华人民共和国法律。

第十三条  合同生效与签署

13.1 本合同自双方法定代表人或授权代表签字并加盖公章之日起
     生效。

13.2 本合同一式四份，甲乙双方各执两份，具有同等法律效力。

13.3 附件清单：
     附件 1：核心研发人员名单
     附件 2：研发资源投入清单
     附件 3：里程碑与交付物明细
     附件 4：技术指标详细说明
     附件 5：增值税专用发票要求

═══════════════════════════════════════════════════════════════

甲方（盖章）：__________  乙方（盖章）：__________
法定代表人/授权代表       法定代表人/授权代表
签字：__________          签字：__________
日期：__________          日期：__________
```

# 模板 B：境外委托研发技术合同（关键差异）

> **使用前提**：受托方为境外法人或自然人。除模板 A 之外，须**额外加入**以下条款：

```
第十四条  跨境合规条款（仅适用境外委托）

14.1 双方确认：本合同项下技术研发涉及跨境技术合作，乙方应
     遵守其所在国家 / 地区的法律法规及国际公约。

14.2 数据出境：涉及甲方业务数据 / 用户数据出境的，应符合中
     华人民共和国《数据安全法》《个人信息保护法》《数据出境
     安全评估办法》要求，必要时甲方先行完成数据出境安全评估。

14.3 出口管制：双方均不得将研发成果或相关技术用于违反任一
     方所在国家 / 地区出口管制法规的用途。

14.4 反贿赂 / 反洗钱：双方应遵守各自所在地反贿赂、反洗钱法
     律法规。

14.5 跨境支付：甲方付款须办理对外支付税款扣缴等手续。乙方
     应配合提供必要的资料（如税收居民身份证明）。

14.6 适用法律变更：本合同适用中华人民共和国法律；争议解决
     地为甲方所在地的有管辖权的法院或中国国际经济贸易仲裁
     委员会（CIETAC）。

第十五条  税务事项（境外委托特别条款）

15.1 双方了解：根据《财政部、国家税务总局、科技部关于完善
     研究开发费用税前加计扣除政策的通知》（财税〔2015〕119
     号）相关规定，**委托境外进行研发活动所发生的费用，
     按照费用实际发生额的 80% 计入委托方的委托境外研发费用；
     委托境外进行研发活动不包括委托境外个人进行的研发活动。**

15.2 双方了解：**委托境外研发费用不超过境内符合条件的研发
     费用三分之二的部分**，可以按规定在企业所得税前加计扣除。

15.3 乙方协助甲方提供研发活动真实性证明，包括但不限于：
     研发人员名单、工作量证明、研发产出物、双方往来邮件 /
     沟通记录等。

第十六条  签署主体特别约定（仅境外）

16.1 本合同中外文版本（如有）与中文版本不一致的，以中文版
     为准。

16.2 双方使用的电子签字（如适用）应符合中华人民共和国《电
     子签名法》及甲方所在地相关法规。
```

> **重要纪律**：根据财税〔2015〕119 号文，**委托境外个人**进行的研发活动**不得享受加计扣除**。所以境外委托对象必须是法人主体（公司、研究机构等）。

# 模板 C：合作研发协议（双方共担风险共享成果）

> **合作研发**与"委托研发"的本质区别：合作研发**双方均投入实质性研发力量**，**共同承担研发风险**，**共享研发成果**。本模板适用于与高校、科研院所、产业链伙伴等的实质性合作。

```
                  合作研发协议

甲方：__________  乙方：__________

第一条  合作背景

双方均具备相应技术研发能力。为开展（项目名称：__________）
研发合作，发挥各自优势，达成本协议。

第二条  研发内容

2.1 项目目标：__________
2.2 甲方承担的研发内容：__________
2.3 乙方承担的研发内容：__________
2.4 双方协同事项：__________

第三条  研发投入

| 投入类别 | 甲方 | 乙方 |
|---------|------|------|
| 资金 | __ 万元 | __ 万元 |
| 人员 | __ 人月 | __ 人月 |
| 设备 / 场地 | __________ | __________ |
| 既有技术 | __________ | __________ |

第四条  ⭐ 知识产权归属

4.1 双方各自带入合作的既有技术成果（"背景技术"），其知识产权
    仍归各自所有。

4.2 合作研发产生的新成果（"前景技术"），其知识产权归甲乙双方
    **共有**。共有方式为：__________
    （选项：① 各自享有平等使用权 ② 按出资比例划分权利份额
    ③ 其他特别约定）

4.3 任何一方不得未经对方书面同意，单独转让、许可前景技术给
    第三方。一方放弃权利的，另一方有优先取得权。

4.4 前景技术的实施或商业化方案，双方另行协商并签订补充协议。

第五条  ⭐ 研发活动性质声明（不可修改）

5.1 双方确认，本协议项下合作研发属于《财政部、国家税务总局、
    科技部关于完善研究开发费用税前加计扣除政策的通知》（财
    税〔2015〕119 号）所定义的"合作研究开发活动"。

5.2 双方各自就其投入部分的研发费用，独立享受加计扣除政策
    优惠。

5.3 本协议将在签订后 30 日内由【任一方 / 双方共同】报主管
    科技行政主管部门进行技术合同登记。

第六条  研发收益分配（如适用）

6.1 前景技术产生的商业收益（含许可费、转让费、产品销售收益
    等），按以下比例分配：
    甲方：__% / 乙方：__%

6.2 收益分配的结算周期：__________

第七条  ~ 第十二条（保密、违约、争议解决等同模板 A）

═══════════════════════════════════════════════════════════════

甲方：__________      乙方：__________
日期：__________      日期：__________
```

# 三套模板对照速查

| 维度 | 模板 A 境内委托 | 模板 B 境外委托 | 模板 C 合作研发 |
|------|----------------|----------------|----------------|
| 受托方 | 境内法人 / 个人 | **仅境外法人**（不含个人）| 双方均为法人 |
| 风险承担 | 委托方承担 | 委托方承担 | 双方共担 |
| 知识产权 | 归委托方 | 归委托方 | 双方共有 |
| 加计扣除入账 | 实际发生额 ×80% | 实际发生额 ×80%（且 ≤境内 2/3）| 各自就投入部分独立享受 |
| 科技局登记 | ✅ 必登 | ✅ 必登 | ✅ 必登 |
| 跨境合规 | 不适用 | ⭐ 必加第 14–16 条 | 视情况 |
| 集中研发分摊 | 不适用 | 不适用 | 可能涉及 |
