## 一、RACI 模型说明

| 字母 | 含义 | 解释 |
|------|------|------|
| **R** | Responsible（负责执行）| 实际执行任务的人，可多人；同一任务可同时多个 R |
| **A** | Accountable（最终问责）| 对任务结果承担最终责任，每项任务**有且仅有一个 A** |
| **C** | Consulted（协助咨询）| 在执行前需被咨询，提供输入 |
| **I** | Informed（知情）| 任务完成后须被通知，不直接参与 |

**重要规则**：
1. 每行（每项任务）必须**有且仅有 1 个 A**
2. R 可以与 A 同一岗位（即"既是执行者也是问责人"）
3. C 与 I 数量不限，但应控制在必要范围内

## 六、特别说明

**1. 工时填报的 A 是员工本人，而不是 PM**
→ 这一项是体系合规性的核心保障：员工对自己填报内容的真实性负终极责任；PM 是复核责任（R + A 用于复核任务 11）。

**2. 月度归集报告的 A 是财务总监而非加计办主任**
→ 月度归集报告对外（财务报表 / 审计）是财务部出口，签发权在财务总监；加计办主任作为最大 R 推动执行。

**3. 税务稽查应对的 A 是 CFO**
→ 涉及对外答复 / 罚款风险 / 公司声誉，必须由 CFO 亲自问责。加计办主任与法务、外部税务师共同执行。

**4. RACI 修订机制**
→ 本矩阵每年汇算清缴后由加计办主任组织复审，提交委员会审议修订。
