| 规则 | 触发 | 处置 |

## 第三章 表 B：6 类汇总表（按月汇总）

### 字段定义

| 字段名 | 类型 | 计算公式 |
|--------|------|----------|

| 第 3 类（保险）金额 | 数字 | SUM 表 A 当月第 3 类金额 |
| 第 4 类（鉴定）金额 | 数字 | SUM 表 A 当月第 4 类金额 |
| 第 5 类（IP）金额 | 数字 | SUM 表 A 当月第 5 类金额 |
| 第 6 类（福利）金额 | 数字 | SUM 表 A 当月第 6 类金额 |
| 第 7 类（差旅）金额 | 数字 | SUM 表 A 当月第 7 类金额 |
| 第 8 类（其他）金额 | 数字 | SUM 表 A 当月第 8 类金额 |
| 其他相关费用总额 | 数字 | SUM 上述 8 类 |
| 研发费用总额 | 数字 | 来自月度归集报告 |
| 当月占比 | 计算 | 其他相关 / 研发总额 ×100% |
| 灯色 | 计算 | 见下文公式 |

### 灯色判断公式

```
IF(占比 < 3.5%, "🟢 安全",
  IF(占比 < 4.5%, "🟡 预警",
    IF(占比 < 5.0%, "🔴 警戒",
      "🚫 超限")
  )
)
```

## 第四章 表 C：占比趋势表（按月计算占比 + 累计 + 预测）

### 字段定义

| 字段名 | 类型 | 计算公式 |
|--------|------|----------|
| 月份 | 文本 | `YYYY-MM` |
| 当月其他相关 | 数字 | 来自表 B |
| 当月研发费用 | 数字 | 来自月度归集报告 |
| 当月占比 | 计算 | 见表 B |
| 累计其他相关（YTD）| 数字 | SUM(1 月-本月)其他相关 |
| 累计研发费用（YTD）| 数字 | SUM(1 月-本月)研发费用 |
| **累计占比（YTD）** | 计算 | 累计其他相关 / 累计研发费用 ×100% |
| 剩余月数 | 数字 | 12 - 当月序号 |
| 预测全年占比 | 计算 | 见下文公式 |
| 预测灯色 | 计算 | 同表 B 灯色规则 |

### 全年占比预测公式

```
预测全年其他相关 = 累计其他相关 + 月均其他相关 × 剩余月数
预测全年研发费用 = 累计研发费用 + 月均研发费用 × 剩余月数
预测全年占比 = 预测全年其他相关 / 预测全年研发费用
```

> **预测的意义**：在 Q2、Q3 提前识别全年是否会超限，给委员会留出 3–6 个月的干预窗口。

## 第八章 自动化规则

### 规则 1：单笔大额预警

- 触发：新增明细记录 + 金额 > 10 万
- 动作：飞书消息 → 加计办主任 + 财务专员 + PM

### 规则 2：月度黄灯通知

- 触发：当月占比 ≥ 3.5%
- 动作：飞书消息 → 加计办主任 + 财务总监；自动启动表 E 处置工单

### 规则 3：月度红灯告警

- 触发：当月占比 ≥ 4.5%
- 动作：飞书消息 → CFO + 委员会全体；启动专题会议召集

### 规则 4：YTD 累计预警

- 触发：YTD 累计占比 ≥ 4.0%
- 动作：飞书消息 → 加计办主任；纳入下次季度评审仪表盘

### 规则 5：预测超限警报

- 触发：预测全年占比 ≥ 5%
- 动作：飞书消息 → CFO；72 小时内出具专题报告

## 第九章 月度更新流程

```
T+1：财务专员录入当月全部明细到表 A
T+2：飞书自动汇总到表 B / C / D（按公式）
T+3：加计办主任复核数据完整性 + 异常项
T+4：自动化规则触发预警（如有）
T+5：仪表盘截图作为月度归集报告附件
T+5：处置工单（如有）启动
```
